Проект интернет-магазин (мини-ERP).
Особенностью данного проекта считаю взятое за основу движение товара на складе и документооборот (ERP).
Основные сущности проекта: 
- DOC (шапка документа)
- DOC_DETAILS (строчки по документу с товаром)
- PRODUCTS_IN_WAREHOUSE (товары на складе)
Основные справочные таблицы с состояниями документов и типами:
- DOCSTATES (состояния)
- DOCTYPES (типы)
- DOC_STATES_TYPES (отражает состояния документов, их порядок, фиксированные состояния)
Основные таблицы с логами:
- HISTORY_PRICES (История цен)
- PRODUCTS_MOVING (История перемещения товара. Фактически отражение движения товара по документам)

Основные типы документов:
- ПН (приходная накладная) - для приходования товара на склад (поступление товара от поставщика)
- РН (расходная накладная) - для списания и отгрузки товара со склада (доставка товара клиенту)
- ЗПок (заявка покупателя) - для заказа товара (интернет-заказ, корзина)
- ЗП (заказ поставщику) - для заявки поставщику на поставку товара в магазин
- ВПок (возврат покупателя) - для возврата товара от клиента в магазин
- ВПст (возврат поставщику) - для возврата товара из магазина поставщику
- Кор (корректировка) - для корректировки складских остатков (приход, расход), изменения цены на товар
